các lệnh thường sử dụng:
git init : Khởi tạo ra Repository Local (.git)
git add <tên_file> : Đẩy các work folder vào trong staging area.
work foder (dự án) | staging area (vùng nhớ kiểm tra trước khi commit (đánh dấu phiên bản)những cái cần thay đổi) |  Repository Local(kho lưu trữ trên máy tính) | repository github kho lưu trữ trên github
